VB-MAPP自理能力检查清单

| 能力  | 年龄  |     |
| --- | --- | --- |
| 穿衣  | **大约18个月** | 摘下帽子  <br>脱下袜子  <br>摘下手套  <br>脱鞋(可能需要帮忙系鞋带、扣鞋扣、扣搭扣)  <br>脱下外套(可能需要帮忙解开纽扣和拉拉链)  <br>拉下裤子(可能需要帮忙解开纽扣和拉拉链)  <br>把裤子提起来(穿着纸尿裤穿裤子时，可能需要帮忙把纸尿裤压紧再穿裤子） |
|     | **大约30个月** | 解开鞋带  <br>解开前面的扣子  <br>解开扣子  <br>系牢和解开鞋扣  <br>解开前拉链(可能很难解开小拉链)  <br>脱衬衫(脱紧身的衬衫可能需要帮忙)  <br>脱裤子或裙子(可能需要帮忙拉开拉链和解开扣子)  <br>穿鞋(需要帮忙区分左右和系鞋带)  <br>穿裤子(可能需要帮忙拉拉链和扣扣子)  <br>整理衣服  <br>能匹配自己的袜子  <br>能匹配自己的鞋  <br>把脏衣服放在篮子里 |
|     | **大约48个月** | 脱衣服(脱紧身套衫但可能需要帮忙)  <br>穿外套  <br>穿袜子  <br>穿裤子  <br>大部分扣子能自己扣上和解开(有的扣子可能很难自己解开和扣上)  <br>拉开和拉上前拉链  <br>扣上和解开前纽扣  <br>扣上和打开前按扣  <br>根据不同的天气状况穿合适的衣服  <br>尝试系鞋花边儿  <br>穿鞋(在帮助下能辨别左右)  <br>尝试系鞋带  <br>把自己的衣服挂在挂钩上  <br>把自己的衣服挂在衣架上(在帮助下)  <br>叠自己的衣服(在帮助下)  <br>把衣服放进抽屉里 |
| 洗澡和梳洗 | 大约18个月 | 用纸巾擦鼻涕(帮助下)  <br>洗手(帮助下)  <br>擦干手(在帮助下)  <br>尝试刷牙(在帮助下) |
|     | 大约30个月 | 尝试在洗澡时使用毛巾和肥皂(在帮助下)  <br>刷牙(在帮助下)  <br>洗脸(在帮助下)  <br>擦干脸  <br>尝试独立洗手  <br>擦干手  <br>尝试梳头(在帮助下) |
|     | 大约48个月 | 用纸巾擦鼻子，然后把它扔进垃圾桶  <br>在较少的帮助下能出入浴缸  <br>洗澡时使用毛巾和肥皂  <br>洗完澡或淋浴后，擦干自己的身体  <br>刷牙  <br>使用牙线清洁牙齿(在帮助下)  <br>洗手  <br>洗脸  <br>擦干脸和双手  <br>洗漱完后把毛巾挂起来  <br>梳头发(在帮助下，特别是较长的头发) |
| 饮食  | 大约18个月 | 用手抓吃食物  <br>用自己的杯子喝水  <br>用勺子舀汤类食物 |
|     | 大约30个月 | 用勺子吃东西  <br>用餐巾擦脸和手  <br>端自己的饭盒或餐盘到餐桌前  <br>打开自己的饭盒  <br>打开密封袋  <br>打开部分食品的包装袋  <br>把一根吸管插入果汁盒  <br>剥香蕉皮  <br>脱下自己的口水巾  <br>用筷子夹起食物（可能需要辅助放进嘴里） |
|     | 大约48个月 | 用勺子的侧面切比较软的食物  <br>使用勺子摊开食物  <br>用筷子夹起食物放进嘴里  <br>吃东西的时候能保持饮食区域的清洁  <br>能打开大多数食品包装  <br>能打开装牛奶或果汁容器  <br>将液体倒入杯子或碗中(从小罐或午餐保温瓶中)  <br>能帮助准备简单的食物(涂抹，搅拌，放入配料)  <br>吃饭时，帮忙摆桌子  <br>把盘子拿到洗涤槽里去  <br>用海绵或抹布擦桌子 |
| 如厕  | 前备技能  <br>大约24个月 | 对强化有反应  <br>听从简单的指令  <br>穿着脏尿布似乎不舒服  <br>每次保持尿布干燥2小时  <br>大便正常，可预测  <br>脱下裤子  <br>把裤子提起来  <br>可以每次静坐2分钟 |
|     | 大约36个月 | 已经学会了使用厕所的单词、符号或PECS(例如，马桶、尿、厕所的符号)  <br>会提要求去上厕所  <br>解开纽扣、打开按扣或拉开裤子拉链  <br>坐在马桶上  <br>用马桶小便  <br>把内裤提起来  <br>把裤子提起来  <br>拉拉链、扣裤子上的暗扣(在帮助下)  <br>冲厕所  <br>洗手(在帮助下)  <br>擦干手 |
|     | 大约48个月 | 站着尿尿时瞄准马桶(男孩)  <br>尿尿后擦拭屁股(女生从前往后擦拭)  <br>拉前拉链  <br>按前纽扣  <br>扣前暗扣  <br>洗手和擦干手 这是如厕的一部分  <br>夜间训练(可能仍有意外) |
